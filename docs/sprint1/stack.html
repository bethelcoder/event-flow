Technology Stack - Sprint 1
Our Project Technology Stack
Component Technology Description
Frontend HTML / EJS Used for structuring pages and dynamic templates for rendering content.
Backend Node.js / Express Server-side logic and API routes handling user requests and database interactions.
Database MongoDB NoSQL database to store users, events, RSVPs, and other project data.
Styling CSS Custom styles to make the interface modern, responsive, and visually appealing.
Version Control Git / GitHub Tracking code changes and collaboration among team members.
Hosting / Deployment Render (project) / GitHub Pages (documentation) The main website is hosted on Render for live access; the documentation site is hosted on GitHub Pages.
